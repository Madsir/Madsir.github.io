UTTERER




Available on the Unity Asset Store



The Utterer package comes with three main components: Utterer Languages, Utterer Dynamics, and Utterer Dialogue System.



Utterer Languages





Utterer Languages localization framework is compatible with Text/TextMesh Pro, AudioSource, and Image components as well as the Utterer Dialogue System.

The system contains 187 languages to choose from (you can also add more) and automatically updates current culture settings based on which language is selected at runtime. Text/UDS translations can be imported and exported as CSV data.




For general use :

Add an UttererLanguages script to any object with a Text, TextMesh Pro, Audio Source, or Image component. Text areas/audio/sprite fields will be added for each language in the list and the Text, TextMesh Pro, Audio Source, or Image component will display the text/audio/sprite of whichever language is the current language in use.

For UDS :

Add an UttererLanguages script to a UDS prefab in the scene. Speaker text areas and utterance text areas will appear in each Utterance for every language in the Languages list, along with an option for localized sound in each Utterance. Text/TextMesh Pro and Utterer Dialogue System translations can be imported and exported as CSV data.








Utterer Dynamics





Built-In (<call>)

Utterer Dynamics contains several built in calls as well as a Dynamics List for you to add your own strings, each with a call and an ID. Calls are replaced with their IDs at runtime.

Built in calls are formatted based on the current culture settings :

Several DateTime calls :

(<LongDate>, <ShortDate>, <LongTime>, etc.)

Currency based on culture settings :

(<c123.4567> = $123.45)

Decimals/doubles/floats, etc. based on culture settings

(<1234.56789> = 1234.568)

Dynamics List :

With the Dynamics List you can create a custom list of calls. As long as at least one UttererDynamics script is in the scene, these calls will be replaced with their ID at runtime in all Text/TMPro instances.

Save custom calls and IDs in Editor

Save custom calls and IDs through user input.


Color coded strings :

If you set the ID of a <call> to "color", it creates a Color Field for you to set a permanent color for the all instances of the text inside the call.






Utterer Dialogue System (UDS)





Utterer Dialogue System lets you easily create and control linear dialogue or subtitles, choose settings, and change appearance all from one place and with no coding necessary. Let the system automatically format itself based on your input or turn off automatic formatting for full customization.



The Utterances array contains the utterances for your dialogue/subtitles, plus content to display for each utterance.



Utterance Content


-Icon of the speaker.
-The speaker text.
-The dialogue/subtitle text.
-Sound played during this dialogue/subtitle.
-Duration the dialogue/subtitle is shown.
-Delay til the next dialogue/subtitle is shown.



UDS Options


Import and Export data from text/.csv files using the Utterer format. Save your dialogue/subtitle sequence or import one from a text file.



UDS Settings


-Delay UDS from starting the dialogue/subtitle sequence.
-Fade the system in and out at the beginning and end of the dialogue/subtitle sequence.
-Fade in and out all content between each utterance.
-Reformat UDS into a subtitle system rather than a dialogue system.
-Set to interactive so UDS progresses dialogue through user input.
-Set to typing animation so UDS adds one letter at a time like a typewriter.
-Customize the size and position of Utterer.
-Use Interactive Input to decide how players progress through dialogue.



UDS Style


-Hide the icon, next button, background, outline, scrollbar, or speaker.
-Choose the color and texture (sprite/image) of each in their respective fields.
-Choose the color, font, font style, font size, etc of the speaker and dialogue/subtitle texts.
