UTTERER




Available on the Unity Asset Store



The Utterer package comes with three main components:
Utterer Languages, Utterer Dynamics, and Utterer Dialogue System.



Utterer Languages





Localization Framework



Compatible with the following components:

Text/TextMesh Pro
AudioSource
Image
Utterer Dialogue System

Contains 187 languages to choose from (you can also add more).

Automatically updates current culture settings based on which language is selected.

Translations can be imported and exported as CSV data.



For General Use :



Add an UttererLanguages script to any object with a Text, TextMesh Pro, Audio Source, or Image component.

Text areas/audio/sprite fields will be added for each language in the list and the component will display the text/audio/sprite of the current selected language.



For Utterer Dialogue System



Add an UttererLanguages script to a UDS prefab in the scene.

Speaker and utterance text areas will appear in each Utterance for every language in the Languages list. Also includes an option for localized sounds in each Utterance.





Utterer Dynamics





Built-In (<call>)



Utterer Dynamics contains several built in calls as well as a Dynamics List for you to add your own strings, each with a call and an ID.

Calls are replaced with their IDs at runtime.

Built in calls are formatted based on the current culture settings.

Examples :

Several DateTime calls :

(<LongDate>, <ShortDate>, <LongTime>, etc.)

Currency :

(<c123.4567> = $123.45)

Decimals/Doubles/Floats :

(<1234.56789> = 1234.568)


Dynamics List



With the Dynamics List you can create a custom list of calls.

As long as at least one UttererDynamics script is in the scene, these calls will be replaced with their ID at runtime in all Text/TMPro instances.



Save Custom Calls & IDs in Editor



Simply add a new element to the Dynamics List in the inspector and customize its call and ID.

The list will automatically save itself when edited.



Save Custom Calls & IDs at Runtime



The Language Chooser prefab saves user input as IDs for custom calls.

Create calls at runtime using the Input's Placeholder text as the call and the user's input as the ID.



Color-coded Strings



If you set the ID of a <call> to "color", it creates a Color Field for you to set a permanent color for the all instances of the text inside the call.






Utterer Dialogue System (UDS)





Create and control linear dialogue or subtitle systems.

Choose from multiple settings.

Change appearance all from one place.

Let the system automatically format itself based on your input or turn off automatic formatting for full customization.

All no coding necessary.

UDS Options



Import and Export data from text/.csv files using the Utterer format.

UDS Settings



Delay UDS from starting the dialogue/subtitle sequence.

Fade the system in and out at the beginning and end of the dialogue/subtitle sequence.

Fade in and out all content between each utterance.

Reformat UDS into a subtitle system rather than a dialogue system.

Set to interactive so UDS progresses dialogue through user input.

Set to typing animation so UDS adds one letter at a time like a typewriter.

Customize the size and position of Utterer.

Use Interactive Input to decide how players progress through dialogue.



UDS Style



Hide the icon, next button, background, outline, scrollbar, or speaker.

Choose the color and texture (sprite/image) of each in their respective fields.

Choose the color, font, font style, font size, etc of the speaker and dialogue/subtitle texts.
