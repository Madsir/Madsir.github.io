Utterer Languages - Localization Framework



Compatible with the following components:

Text/TextMesh Pro
AudioSource
Image
Utterer Dialogue System

Contains 187 languages to choose from (you can also add more).

Automatically updates current culture settings based on which language is selected.

Translations can be imported and exported as CSV data.






Utterer Dialogue && Utterer Captions Systems



Create linear or dynamic dialogue/captions using Utterer's visual dialogue tree.

Control which content to display using multiple settings.

Easily customize the system's appearance.

Fade dialogue in and out, use typing animations, control user input. All this with no coding necessary.





Utterer Dynamics - Dynamic String Interpolation



Create a list of custom calls.

Create color-coded string calls.

Save custom calls & IDs in Editor and at Runtime.

Calls are replaced with their IDs at runtime.
