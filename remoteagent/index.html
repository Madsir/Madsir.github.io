REMOTE AGENT





Far out in the remote wilderness lives an agent whose aerospace monitoring station has been attacked by an unknown source.
Left with no means of communication, he is forced to leave his station and seek help.
As his journey begins, however, it becomes clear that whatever source destroyed his monitoring station has also unleashed a strange, new chaos across the land.














Tune in to Remote Agent Radio for Remote Agent footage, info, and updates, as well as music by Toilet Noise.
Each broadcast contains a morse code message.
Translate the morse code and use the answer as a key to solve the cryptogram in the video's description.
Each solved cryptogram reveals a little more about Remote Agent.
