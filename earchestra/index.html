EARCHESTRA




[In Production]


Earchestra is a fresh and versatile audio manager for Unity Engine.



Sections


Earchestra has three main sections: Music, SFX, and Speech.
- Create playlists and add tracks in each of these sections
- Customize your playlist and track names.
- Control playlists using Earchestra's Maestro methods.
- Save or load a playlist collection.



Playlists


- Play in index order or mixed order.
- Fade tracks at the beginning and end.
- Control how much delay between tracks



Volumes


Control the volumes for each section. Includes a prefab slider system that allows the player to set the master volumes for each section. The system saves and loads these volume values automatically.



Audio Army


Build an audio army capable of handling more complex and dynamic audio demands.

- Play multiple playlists from the same section at the same time.
- Play multiple sounds at once, each with their own effects.
- Make any clip from your playlists play at the Transform of your choice.



Earchestra's Maestro Methods


Earchestra has multiple available methods for each Section as well as master commands for the entire system.
- Control playlists, tracks, effects, and transforms.

- Use these methods in any Monobehavior scripts.
