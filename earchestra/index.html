--- layout: default title: Earchestra description: "Earchestra is a command-based sound manager for Unity Engine." ---



Create audio playlists and control them by commanding Earchestra's built-in Maestro.

Earchestra's Maestro Methods can be called from any MonoBehavior script.

Available on the Unity Asset Store


Sections :
Music, SFX, Speech


Create playlists and add tracks in each of these sections.

Customize your playlist and track names.

Control playlists using Earchestra's Maestro methods.



Ensemble


Play multiple playlists from the same section at the same time.

Play multiple tracks from a Section's playlist, each with their own transform.

Make any clip from your playlists play at the position of your choice.



Earchestra's Maestro Methods


Control playlists, tracks, and transforms.

Use these methods in any Monobehavior scripts.
