EARCHESTRA


[In Production]



Create audio playlists and control them from any script with this simple sound manager.


Earchestra has three main sections: Music, Sound Effects, and Speech.
You can create an infinite amount of playlists in each of these sections and control them from any of MonoBehavior script using Earchestra's Maestro functions and commands.
Earchestra has the ability to summon an audio source army to "throw its voice", meaning you can make any clip from your playlists play at the Transform of your choice.


Music Playlists


- Each playlist can play in index order or mixed order.
-Each playlist can fade tracks at the beginning and end.
-Each track in a music playlist can be set to loop

Commands : (For use with Earchestra's Maestro functions)

MixOn - plays specified playlist's tracks in random order
MixOff - plays specified playlist's tracks in index order
Play(string) - plays the named track in the specified playlist
Play(int) - plays the numbered track in the specified playlist
Play - begins playing the specified playlist
Pause - pauses current track
Unpause - unpauses current track
Stop - stops the current track
Next - plays next track in specified playlist
Restart - restarts specified playlist
LoopOn(string) - loops the named track in the specified playlist
LoopOn(int) - loops the numbered track in the specified playlist
LoopOff(string) - turns off loop for the named track in the specified playlist
LoopOff(int) - turns off loop for the numbered track in the specified playlist



Sound Effects Playlists


Play a named sound effect or random sound effect from the specified SFX playlist
Play the sound from a specified transform using Earchestra's Audio Army



Speech Playlists

Play a named speech sound or random speech sound from the specified speech playlist
Play the speech sound from a specified transform using Earchestra's Audio Army
