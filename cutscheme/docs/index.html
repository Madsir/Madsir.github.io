Cutscheme VFX Systems


Cutscheme Effect System
(Universal Render Pipeline only)



The Cutscheme Effect System is fully integrated into Cutscheme 2D and Cutscheme 3D. It will enable automatically when an effect is selected in a Camera Shot and display the effect(s) at runtime during that Camera Shot.

The System makes a copy of the Scene Volume and uses the copy during the cutscene. After the cutscene, the system re-enables the original Scene Volume and removes the copy.

Display multiple camera effects during each shot in the cutscene.

Customize the override values for the copy's Vignette, Bloom, Film Grain, and Color Filter.






Cutscheme Filter System
(Built-In Renderer Pipeline only)



The Cutscheme Filter System is fully integrated into Cutscheme 2D and Cutscheme 3D. It will enable automatically when a filter is selected in a Camera Shot and display the filter(s) at runtime during that Camera Shot.

Display up to two camera filters at a time during each shot in the cutscene.

Customize with your own filters (Materials) or use the ones that come with the package.
