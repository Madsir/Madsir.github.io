Studio in the Woods




Madsir Studio presents original games, game development assets, and artwork created by artist/solo-developer Hillary Bridget Bird.

Regularly collaborates with musician/composer CR Bird's Toilet Noise.


Missions:



Create fresh and modern games that still retain the old-school flavor of games from the 20th century.

Create a variety of asset packages intended to make game development less time consuming.
