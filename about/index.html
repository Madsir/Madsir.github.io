STUDIO IN THE WOODS

MAKING GAMES

On an adventure to create fresh, modern games that still retain that old-school flavor of games from the 20th century: unforgiving and tough, like real GAME OVERs when you run out of lives.

BUILDING TOOLS

Creating a style of development tools which do the tedious work for you by automatically formatting operations to match your input, allowing you to add creative elements to your projects without all the time-consuming scripting.
